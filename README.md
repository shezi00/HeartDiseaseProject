Heart Disease Prediction App

A Machine Learning web app built with Streamlit that predicts the risk of heart disease based on patient health data.

Features
- User-friendly interface
- Real-time prediction
- ML model (KNN) used for classification

Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib

Run Project
```bash
pip install -r requirements.txt
streamlit run app.py

Files
app.py → Streamlit app
KNN_heart.pkl → trained model
scaler.pkl → data scaler
columns.pkl → feature columns

Note:
This is for educational purposes only, not medical advice.
